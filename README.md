# Информационная система «КультРяд»
Система учета и продажи билетов на культурные мероприятия.

## Описание проекта
Данная система автоматизирует процесс продажи билетов, управления базой данных мероприятий и контроля доступа посетителей по QR-кодам.

## Технологический стек
- **Моделирование:** SADT (IDEF0, IDEF3, DFD), UML 2.5 (Visual Paradigm / Mermaid).
- **Проектирование:** ГОСТ 34.602-2020, Figma.
- **Разработка:** C++, Visual Studio (WinForms / MFC).
